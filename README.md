# Hello World – Node.js HTTP Server

A minimal HTTP server built with Node.js's built-in `http` module.
Returns "Hello, World!" for every request.

## File

hello-world.js

## How to Run

node hello-world.js

Server will start at: http://127.0.0.1:3000/

## How to Test

Open browser and visit:
http://127.0.0.1:3000/

Or use curl in terminal:
curl http://127.0.0.1:3000/

## How to Stop

Press Ctrl+C in the terminal.

## Requirements

- Node.js (any modern version)
- No external packages needed
